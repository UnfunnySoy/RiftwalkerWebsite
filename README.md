Project Title: Riftwalker Website - Website and Data
Framework: ASP.NET MVC


The system has been developed and tested on the following platforms:
- Windows 10 / 11
- Other platforms supported by ASP.NET may work but were not tested.

Implemented Programming Language
- C#
The program also uses HTML and Razor to render views.

Required Libraries and Dependencies
- .NET 9.0
- .NET 9.0 Targeting
The project uses many pre-included MVC modules provided by the framework.
No external packages or libraries required at this stage.

Steps to Run the System
1. Install Visual Studio
2. Download or clone the project repository.
3. Open the project solution in Visual Studio.
4. Run the project. The website should open on its own.
  4a.) If the website does not open on its own, check the output for the port the website is running on.
  4b.) Open your preferred browser and navigate to "localhost:<YOUR_PORT_NUMBER>". The website should load
5. You should be able to navigate the website using the toolbar at the top of the page.

Other Notes
- The current build implements:
  - Save data recovery with the client
  - Leaderboard data save and communication
  - Data storage for user accounts and run data
